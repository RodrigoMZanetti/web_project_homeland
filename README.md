# Tripleten web_project_homeland

Este projeto foi desenvolvido como parte do curso de Desenvolvimento Web da TripleTen.  
O objetivo é criar uma página responsiva e acessível que apresenta a galeria de arte da TripleTen, com citações inspiradoras e informações sobre profissionais da comunidade.

## Funcionalidades

- Exibe o título e subtítulo principais da galeria
- Mostra imagens e citações em seções separadas
- Apresenta texto informativo sobre os artistas
- Adapta o layout para diferentes tamanhos de tela (320px, 544px, 1024px e superiores)
- Inclui rodapé com símbolo de copyright

## Tecnologias utilizadas

- HTML5
- CSS3 (Flexbox e Media Queries)
- Normalize.css
- Fonte Inter (Google Fonts)
